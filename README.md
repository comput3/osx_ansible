# osx_ansible

Automated personalized configuration of OSX. 

## Installation

Use the bootstrap.sh to deploy the ansible configurationa, and run the app

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
